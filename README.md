# F.CSM306-Lab02-Starter
**Зорилго:** Тус лабораторийн ажлын үндсэн зорилго нь олон цөмт (Multi-core) процессорын тооцооллын нөөцийг оновчтой удирдах, өндөр гүйцэтгэлтэй системийг загварчлан хөгжүүлэхэд оршино. Үүнд дараах даалгаваруудыг хэрэгжүүлнэ. 

**1. TaskSystemParallelSpawn** (класс) хэрэгжүүлэлт: run() функц ажиллах болгонд үйлдлийн системээс thread үүсгэж, тухайн ажил дуусмагц устгана. Нийт даалгавар num_total_tasks thread-үүдэд хуваарилна. Ажлыг статикаар хувиарлана. Бүх thread ажлыг хийж дууссаны дараа буцаж үндсэн thread-тэй join() хийж нийлүүлнэ үү.  (Энэ даалгаврын хувьд ямар сул тал байна вэ?)

**2. TaskSystemParallelThreadPoolSpinning** (класс) хэрэгжүүлэлт: Энэ хэсэгт thread үүсгэх зардлыг арилгахын тулд (TaskSystemParallelSpawn) thread pool ашиглана. Өөрөөр хэлбэл програмын эхэнд байгуулагч функц дотор thread-үүдийг ганц удаа үүсгээд, програмыг дуустал устгахгүйгээр дахин ашиглана. Ямар нэг thread – ийн ажил дууссан бол while(true) {} давталтанд ажил ирэхийг хүлээнэ (spinning). Хоёр thread нэг ижил дугаартай ажлыг зэрэг авч хийхгүйн тулд std::mutex ашигла. (Энэ даалгаврын хувьд ямар сул тал байна вэ?)  

**3. TaskSystemParallelThreadPoolSleeping** (класс) хэрэгжүүлэлт: CPU-ийн нөөцийг зөвхөн хэрэгтэй үед ашиглаж нөөцийг хэмнэнэ. (Spinning-тэй харьцуулахад)  Thread-үүд ажилгүй үедээ түр унтах горимд шилжинэ (wait). Шинэ ажил ирэхэд Мастер thread унтаж буй ажилчдыг сэрээнэ. Бүх ажил дууссаны дараа сүүлчийн ажилчин Мастер thread-ийг сэрээнэ. Үүнийг хэрэгжүүлэхийн тулд std::condition_variable (нөхцөлт хувьсагч)-ийг ашиглаарай.

Програм ажиллаж дуусахад дараах байдлаар үр дүн гарна. Хэрэв хэрэгжүүлэлтийг зөв хийсэн бол Parallel Sleeping Pool нь хугацааны хувьд хамгийн бага хугацаанд ажиллана.  Serial > Parallel Spawn > Parallel Spinning Poos > Parallel Sleeping Pool. 

```
Testing [Serial System]... Done. Time: 0.0172s
Testing [Parallel Spawn]... Done. Time: 0.0024s
Testing [Parallel Spinning Pool]... Done. Time: 0.0016s
Testing [Parallel Sleeping Pool]... Done. Time: 0.0015s
```

 
**main.cpp:** Туршилтыг ажиллуулж, хугацааг хэмжих үндсэн файл.

**itasksys.h:** Энэ нь бүх Task System-үүдийн суурь интерфейс (Abstract Class) ба энд run() болон sync() гэсэн виртуал функцуудыг тодорхойлсон. Оюутны хэрэгжүүлж буй бүх классууд үүнээс удамшиж ажиллана.

**tasksys.h (Header):** Лабораторийн заавар дээрх 4 өөр системийг (Serial, Spawn, Spinning, Sleeping) класс болгон энд тодорхойлсон.

**tasksys.cpp (Implementation):** Дээрх классуудын логик буюу "биеийг" энд бичнэ. Thread үүсгэх, Mutex ашиглан түгжих, Condition Variable-оор сэрээх зэрэг С++ кодыг энд хэрэгжүүлнэ үү.
Даалгаврыг 6 долоо хоногт шалгана.
